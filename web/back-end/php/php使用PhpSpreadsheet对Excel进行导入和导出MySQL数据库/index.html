# 初识 PhpSpreadsheet

PhpSpreadsheet 是一个开源的 PHP 库，可以用来处理各种电子表格文件，包括 Excel、CSV、ODS 等。
官网：https://phpspreadsheet.readthedocs.io/en/stable/
